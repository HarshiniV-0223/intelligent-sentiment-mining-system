# Intelligent Sentiment Mining System for E-Commerce Platform

## Project Overview
This project is an AI-powered sentiment analysis and recommendation system for e-commerce platforms. It analyzes customer reviews using Machine Learning and Natural Language Processing (NLP) to classify sentiments as Positive, Negative, or Neutral. The system also provides product recommendations and visual insights through an interactive Streamlit dashboard.

## Technologies Used

- Scikit-learn
- Streamlit

## How to Run

### Install Required Libraries
```bash
pip install -r requirements.txt
```

### Run the Application
```bash
streamlit run app.py
```

## Project Outcome
This project helps users understand customer opinions, analyze review sentiments, identify trends, and receive product recommendations. It demonstrates the practical application of Machine Learning and NLP in the e-commerce domain.

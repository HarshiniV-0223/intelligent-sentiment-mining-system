# Intelligent Sentiment Mining System for E-Commerce Platform

## Project Overview
This project analyzes customer reviews from e-commerce platforms and predicts whether the sentiment is Positive, Negative, or Neutral using Machine Learning and Natural Language Processing (NLP).

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib
- Plotly
- Joblib

## Features
- Sentiment Prediction
- Customer Review Analysis
- Data Visualization
- Interactive Streamlit Dashboard
- Machine Learning Based Classification

## Files Included
- app.py
- train_model.py
- utils.py
- model.pkl
- vectorizer.pkl
- requirements.txt
- Womens Clothing E-Commerce Reviews.csv

## How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py

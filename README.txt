PROJECT TITLE:
Advanced Intelligent Sentiment Mining and Business Insight System for E-Commerce Platforms

DESCRIPTION:
This project analyzes customer reviews from E-Commerce platforms such as Amazon, Flipkart, Meesho, and Myntra.
It performs intelligent sentiment mining and converts customer feedback into business insights.

KEY FEATURES:
1. Upload any E-Commerce CSV dataset
2. Auto-detect important columns
3. Sentiment Analysis (Positive / Negative / Neutral)
4. Product-wise sentiment analysis
5. Seller-wise performance analysis
6. Brand-wise performance analysis
7. Complaint keyword mining
8. Positive feedback mining
9. Product improvement recommendations
10. Download analyzed dataset

UNIQUE CONTRIBUTION:
This project goes beyond normal sentiment analysis by identifying:
- Why products receive negative feedback
- Which products need improvement
- Which sellers/brands are underperforming
- What business actions should be taken

TOOLS USED:
- Python
- Streamlit
- Pandas
- Plotly
- TextBlob
- Matplotlib
- WordCloud
